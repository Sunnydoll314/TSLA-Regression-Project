# TSLA Regression Project

This project predicts Tesla (TSLA) stock closing prices using regression models.

## Models
- Multiple Linear Regression  
- Ridge Regression  
- Lasso Regression  

## Dataset
Tesla stock data (2010–2020)

## Tools
- Python  
- Pandas  
- Scikit-learn  
- Matplotlib  

## Result
The model achieved very high accuracy (R² ≈ 0.999).  
However, it contains look-ahead bias because same-day features were used.
